# social-recruiter-django-vuex
## Description
Recruiting Service using Twitter

## Requirement
- BackEnd : Django
  - version: 2.1
- FrontEnd : Vue.js
  - version: 2.5.16

## Install
- FrontEnd
```
$ cd client
$ npm install
$ npm run webpack
```

- BackEnd
```
$ cd backend
$ python manage.py migrate
$ python manage.py createsuperuser
$ python manage.py runserver
```
